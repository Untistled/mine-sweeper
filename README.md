## mine-sweeper

这是一个用 Rust 编写的命令行扫雷游戏项目。以下是这个项目的结构和功能：

### 2 代码结构

#### 2.1 主程序 (main.rs)

提供简单的菜单界面，支持三种模式：

- Default: 默认游戏 (9x9 网格，10 个地雷)
- Customize: 自定义游戏 (可设置宽度、高度和地雷数量)
- Exit: 退出游戏

#### 2.2 数据模块 (data.rs)

##### 2.2.1 输入处理模块 (input)

- str(): 获取字符串输入
- uint8() / uint16(): 获取无符号整数输入
- opt(): 显示选项菜单并获取选择
- 核心数据结构
- Cell (单元格)

##### 2.2.2 状态 (State):

- Covered(Tag): 被覆盖 (三种标记: Default *, Determined !, Undetermined ?)
- Discovered: 已揭开

##### 2.2.3 对象 (Object):

- Mine: 地雷
- Null(u8): 空单元格，数字表示周围地雷数量
- Table (游戏板)

##### 2.2.4 二维 Cell 向量

- 负责生成随机地雷位置
- 计算每个单元格周围的地雷数量
- 处理点击和标记操作

##### 2.2.5 Game (游戏)

- 封装 Table，提供游戏逻辑
- 处理用户操作和游戏循环

### 3 游戏机制

#### 3.1 操作 & 反应:

- x<列字母><行数字>: 揭开单元格
- s<列字母><行数字>: 切换标记状态
- 自动展开: 揭开数字为0的单元格时自动展开周围单元格
- 胜利条件: 所有非地雷单元格都被揭开
- 失败条件: 揭开地雷
